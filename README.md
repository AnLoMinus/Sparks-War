## 📖 תקציר הפרויקט

מלחמת הניצוצות הוא פרויקט מולטימדיה המתאר מאבק אפי בין כוחות האור לכוחות הרשע. במרכז העלילה עומד הרעיון שבכל אדם קיים ניצוץ ייחודי, והמאבק מתנהל על שליטה בניצוצות אלו ועל עתיד האנושות.

- **🌸 חן - מגינת האור**: לוחמת בעלת יכולת לראות ולחזק ניצוצות

- **⚡ חיל טכנולוגיית האור**: מפתחי טכנולוגיות קדושות
