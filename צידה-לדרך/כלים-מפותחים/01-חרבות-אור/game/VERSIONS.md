# 📌 VERSIONS - ניהול גרסאות

## מדיניות גרסאות

הפרויקט עוקב אחר [Semantic Versioning](https://semver.org/lang/he/):

```
MAJOR.MINOR.PATCH
```

- **MAJOR** (גרסה עיקרית) - שינויים שאינם תואמים לאחור
- **MINOR** (גרסת משנה) - תכונות חדשות תואמות לאחור
- **PATCH** (תיקון) - תיקוני באגים תואמים לאחור

## 📥 הורדה והתקנה

### גרסה נוכחית (0.0.1)

**דרך 1: Clone מ-Git**
```bash
git clone https://github.com/sparks-war/light-swords-game.git
cd light-swords-game/game
```

**דרך 2: הורדה ישירה**
1. הורד ZIP מהמאגר
2. חלץ את הקבצים
3. פתח `index.html` בדפדפן

**דרך 3: Live Server**
```bash
# עם Python
python -m http.server 8000

# עם Node.js
npx http-server
```
