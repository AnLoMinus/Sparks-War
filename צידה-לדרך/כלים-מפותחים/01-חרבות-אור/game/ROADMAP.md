# 🗺️ ROADMAP - מפת דרכים

## חזון המוצר

**משחק חרבות אור** נועד להיות משחק אימון אינטראקטיבי, חינוכי וכיפי המלמד את השחקנים את עקרונות מלחמת הניצוצות תוך כדי משחק מהנה ומאתגר.

**ערכי הליבה**:
- 🎯 פשוט ללמוד, מאתגר לשלוט
- ⚡ פעולה מהירה וריתמית
- 🧠 אסטרטגיה ותכנון
- 🌟 חינוך רוחני דרך משחק
- 🤝 קהילה ושיתוף
